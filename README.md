# Mapshaper

[Repositorio](https://github.com/mbloch/mapshaper)

Instalar mapshaper:

```npm install mapshaper -g --user```

Comprimir mapa con mapshaper

```mapshaper -i fichero.geojson snap -simplify 0.1 -o force fichero.geojson```
